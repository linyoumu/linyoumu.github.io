总结：
如果自己的Target编译调试遇见问题，尝试以下配置：
1: enable hardened runtime -> NO
2: build phase -> denpendenice -> objc
3: 如果main函数中断点断不住，尝试重新创建自己的Target将main放在Compile Sources最前面。
